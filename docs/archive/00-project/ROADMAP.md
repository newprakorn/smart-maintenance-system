# 1. Project Vision

พัฒนาระบบ Smart Maintenance System สำหรับโรงงานอุตสาหกรรมที่สามารถใช้งานจริง รองรับงานซ่อมบำรุง งาน PM การติดตามเครื่องจักรผ่าน QR Code และสามารถขยายไปสู่ระบบ Inventory, IoT และ AI ได้ในอนาคต
