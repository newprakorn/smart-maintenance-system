# Project Vision

Develop a modern CMMS (Computerized Maintenance Management System)
for manufacturing plants using

- React
- TypeScript
- Supabase
- PostgreSQL

The system is designed to support

- Preventive Maintenance
- Corrective Maintenance
- Inventory Management
- Work Order Management
- Reporting
- Dashboard
- User & Role Management
