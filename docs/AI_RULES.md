# AI Development Rules

## General

* Use TypeScript Strict Mode.
* Never use `any`.
* Prefer reusable components.
* Keep components small and focused.
* Use Feature-Based Architecture.

## Naming

* Folder: kebab-case
* Component: PascalCase
* Variable: camelCase

## React

* Functional Components only.
* Use Hooks.
* Avoid duplicated code.

## Database

* Prisma ORM
* PostgreSQL
* UUID Primary Key

## Code Style

* Readability first.
* Type Safety first.
* Clean Architecture.
* SOLID Principles.

## Goal

Generate production-ready code suitable for a factory environment.
